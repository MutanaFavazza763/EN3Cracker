解除希沃白板3的15天试用期结束后的限制

### 声明
本项目仅提供激活所需的修改后的希沃白板二进制文件以及本程序，**不提供任何有关希沃白板本体的源代码**！！！  
希沃白板3著作权归希沃官方（广州视睿电子科技有限公司、广州视源电子科技股份有限公司）所有。

## 项目背景
众所周知希沃白板3已经被官方抛弃，而希沃白板5砍掉了许多希沃白板3中有的功能，因此我们班决定同时在电脑上安装希沃白板5与3。而打开希沃白板3却发现需要激活，班主任说班费不差钱，问题是希沃官方已经把旧版的激活服务器关停了，也就是你有正版激活码也没法激活了（希沃怎么辣么坏啊TAT）度娘一番激活/破解教程无果，无奈之下决定从软件逆向入手。

## 原理
通过逆向工程修改希沃白板的激活组件，使其在被调时用无论如何都返回已激活状态。
